# FTP server

Python version: 3.9.6

| Variable         | Value                                            |
|------------------|--------------------------------------------------|
| url              | server url (ex. https://ftp-serv.herokuapp.com)  |
| database         | Database name                                    |
| host             | Database host                                    |
| password         | Database password                                |
| port             | Database port                                    |
| user             | Database user                                    |
